Help

You can get help tips for commands usage.

$ banga help $ banga <command> --help

Commands

Here's a list of available commands

command Alias Description
docs d Opens BÃ ngÃ¡ documentation in a browser
generate g Generates and/or modifies files.
help h Lists available commands and their descriptions.

docs

Opens BÃ ngÃ¡ documentation in a browser

$ banga docs $ banga d

generate

Generates and/or modifies files

$ banga generate <type> <name> [options] $ banga g <type> <name> [options]
Example:
$ banga generate controller user --no-crud $ banga g asset user $ banga generate service user $ banga g r user $ banga generate m user

Types

Type Alias Description
asset a Create controller, model, route and service files.
controller c Create controller file
model m Create model file
route r Create route file
service s Create service file

Options

Option Description
--auth Add auth middleware to routes
--no-controller Do not create controller file
--no-create Do not include create methods
--no-crud Do not include CRUD methods
--no-delete Do not include delete methods
--force Replace file if it already exists
--no-model Do not create model file
--no-read Do not include read methods
--root Creates file at the root of the current working directory
--no-route Do not create route file
--no-service Do not create service file
--no-update Do not include update methods
Note:
  • The following options --no-controller, --no-model, --no-route and --no-service are only avilable for asset type
  • You can also specify a path to nest your feature within subdirectories. For example, a name of "users/model" will generate the model file at src/users/model.

help

Lists available commands and their short descriptions.

$ banga help
$ banga h

new

Creates a new Express app.

$ banga new <name> [options] $ banga n <name> [options]
Example: $ banga new hello-app --no-auth $ banga new myapp

Options

Options Description
--force Replace file if it already exists
--no-auth Do not include User and authentication files
--no-dep Do not install packges
--no-git Do not initialize git
